# PersonalProject

This repository is a planning and research workspace for a graduate-level AI portfolio focused on computer vision, transformers, and applied machine learning. It collects project ideas, dataset notes, and early system-design thinking for future implementation work.

## Project Focus

The current direction centers on three major areas:

- Object detection with CNN, R-CNN, and transformer-based models
- Multimodal image-text learning inspired by CLIP-style systems
- A domain-specific LLM with retrieval-augmented generation for a mechanic assistant

There is also a strong emphasis on medical AI, with **MedMNIST** marked as a high-priority dataset for experimentation and portfolio development.

## Repository Structure

- `README.md` - high-level description of the repository
- `Overview/project_ideas.md` - proposed AI project directions and rough development phases
- `Overview/data_sources.md` - recommended datasets for learning CNNs and transformers
- `Overview/data/` - placeholder directory for supporting data-related files

## Current Status

This repository is currently in the planning stage. It does not yet contain model training code, experiments, or deployment artifacts. The main purpose right now is to organize ideas, choose datasets, and define a practical roadmap for implementation.

## Planned Architecture Themes

The notes in this repository suggest a full-stack AI application direction that may eventually include:

- A user interface layer
- An application/services layer
- A system logic layer
- A database layer, potentially using MongoDB and AWS-backed infrastructure

## Next Steps

Likely next steps for this project include:

1. Select one priority project to build first
2. Choose a starter dataset such as CIFAR-10 or MedMNIST
3. Set up the training environment and project structure
4. Implement baseline models and evaluation pipelines
5. Expand into deployment, UI, and system integration

## Goal

The goal of this repository is to grow into a strong applied AI portfolio that demonstrates both machine learning depth and practical software engineering skills.
